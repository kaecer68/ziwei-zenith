### 十二宮位

逆時針順序：命宮 → 兄弟宮 → 夫妻宮 → 子女宮 → 財帛宮 → 疾厄宮 → 遷移宮 → 僕役宮 → 官祿宮 → 田宅宮 → 福德宮 → 父母宮

| 宮位 | 代表意義 |
|------|----------|
| 命宮 | 自我、性格、命運基調 |
| 兄弟宮 | 兄弟手足、同輩關係 |
| 夫妻宮 | 婚姻、感情、配偶 |
| 子女宮 | 子女、晚輩、興趣 |
| 財帛宮 | 財富、收入、理財 |
| 疾厄宮 | 健康、體質、疾病 |
| 遷移宮 | 外出運、人際印象 |
| 僕役宮 | 朋友、下屬、社交圈 |
| 官祿宮 | 事業、工作、地位 |
| 田宅宮 | 房產、家庭、祖業 |
| 福德宮 | 精神、福氣、享受 |
| 父母宮 | 父母、長輩、學歷 |

### 排盤算法步驟

1. **取得出生時間**：農曆年、月、日、時（精確到時辰）
2. **安命宮、身宮**：
   - 從寅宮起正月，順數至出生月
   - 從該宮起子時，逆數至出生時 → 命宮
   - 順數至出生時 → 身宮
3. **安十二宮**：以命宮為起點，逆時針排列
4. **定五行局**：根據年干支和命宮地支確定（水二局、火六局、木三局、土五局、金四局）
5. **安紫微星**：公式計算出生日 ÷ 五行局數
6. **安十四主星**：以紫微星為基礎，按口诀依次安置其餘十三顆主星
7. **（進階）安輔星、煞星、四化星**
