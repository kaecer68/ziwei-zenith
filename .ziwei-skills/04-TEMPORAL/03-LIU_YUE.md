# 流月計算

> **前置要求**：[流年計算](./02-LIU_NIAN.md)

---

## 1. 流月基礎

### 1.1 什麼是流月？

流月 = 每月的運勢，以**農曆月**計算。

### 1.2 流月地支

| 農曆月 | 地支 |
|:-------:|:----:|
| 正月 | 寅 |
| 二月 | 卯 |
| 三月 | 辰 |
| ... | ... |

### 1.3 流月天干

根據「流年干 + 月份」推算：

| 流年天干 | 正月起月干 |
|:--------:|:-----------:|
| 甲、己 | 丙 |
| 乙、庚 | 戊 |
| 丙、辛 | 庚 |
| 丁、壬 | 壬 |
| 戊、癸 | 甲 |

---

## 2. 流月計算流程

### Step 1: 確定流月地支

流月地支 = 根據節氣確定的月份地支

### Step 2: 確定流月天干

```
流月天干 = 流年起月干 + (月份 - 1) % 10
```

### Step 3: 流月落宮

流月落宮 = 流月地支確定的宮位

---

## 3. 流月四化

```
流月四化 = 流月天干 → 四化表 → 飛星四化
```
