# 流日計算

> **前置要求**：[流月計算](./03-LIU_YUE.md)

---

## 1. 流日基礎

### 1.1 什麼是流日？

流日 = 每日的運勢，以**農曆日**計算。

### 1.2 流日地支

流日地支 = 根據日期確定的地支（60甲子循環）

### 1.3 流日天干

流日天干 = 根據日期確定的天干（60甲子循環）

---

## 2. 流日計算

使用 **lunar-zenith** 套件計算精確的流日干支。

### 流日落宮

```
流日落宮 = 流日地支確定的宮位
```

---

## 3. 流日四化

```
流日四化 = 流日天干 → 四化表 → 飛星四化
```

---

## 4. 動態運限層級總結

```
本命盤（生年四化）
    │
    ├── 大限（10年）→ 大限四化
    │       │
    │       └── 流年（1年）→ 流年四化
    │              │
    │              └── 流月（1月）→ 流月四化
    │                     │
    │                     └── 流日（1日）→ 流日四化
```

---

## 5. 顯示順序

當查看某一層級時，應同時顯示該層級以及其**上層**的四化：

| 當前層級 | 顯示的四化 |
|---------|-----------|
| 本命 | 生年四化 |
| 大限 | 生年四化 + 大限四化 |
| 流年 | 生年四化 + 大限四化 + 流年四化 |
| 流月 | 生年四化 + 大限四化 + 流年四化 + 流月四化 |
| 流日 | 生年四化 + 大限四化 + 流年四化 + 流月四化 + 流日四化 |
