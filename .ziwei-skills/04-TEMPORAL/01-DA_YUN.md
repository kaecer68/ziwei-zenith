## 1. 大限基礎

### 1.1 什麼是大限？

大限是人生每10年一個階段的運勢，以年齡劃分。

### 1.2 大限年齡計算

| 大限 | 年齡範圍 |
|:----:|:---------:|
| 第1大限 | 0-9歲 |
| 第2大限 | 10-19歲 |
| 第3大限 | 20-29歲 |
| ... | ... |

### 1.3 大限計算公式

```
大限序號 = floor(年齡 / 10)
大限起始年 = 出生年 + 大限序號 * 10
```

---

## 2. 大限宮位

### 2.1 大限落宮

大限不是固定在某個年，而是10年整體對應一個宮位。

### 2.2 大限宮位計算

```
大限宮位 = (命宮地支Index + 大限序號 * 1) % 12
```

### 2.3 大限宮干

大限宮位的天干 = 該宮位地支對應的天干

---

## 3. 大限四化

### 3.1 大限四化的意義

大限期間，該大限宮干激活的四化飛星。

### 3.2 大限四化計算

```
大限四化 = 大限宮干 → 四化表 → 飛星四化
```

---

## 4. 大限與流運的關係

```
本命盤（基礎）
    │
    ├── 大限（10年為一單位）
    │       │
    │       └── 流年（1年為一單位）
    │              │
    │              └── 流月（1月為一單位）
    │                     │
    │                     └── 流日（1日為一單位）
```
