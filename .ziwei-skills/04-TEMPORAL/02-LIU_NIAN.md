# 流年計算

> **前置要求**：[大限計算](./01-DA_YUN.md)

---

## 1. 流年基礎

### 1.1 什麼是流年？

流年 = 每年的運勢，以**民國年**計算。

### 1.2 流年地支

流年地支 = 根據立春後的**年份地支**：

| 年份地支 | 流年宮位 |
|:--------:|:---------:|
| 子年 | 子宮 |
| 丑年 | 丑宮 |
| 寅年 | 寅宮 |
| ... | ... |

### 1.3 流年天干

流年天干 = 該年的天干

---

## 2. 流年計算流程

### Step 1: 確定流年地支

```
流年地支 = (出生年地支 + (目標年 - 出生年)) % 12
```

### Step 2: 確定流年天干

根據「五虎遁」起月干，然後推算年干...

（簡化：直接使用該年的天干）

### Step 3: 流年落宮

流年落宮 = 流年地支確定的宮位

---

## 3. 流年四化

```
流年四化 = 流年天干 → 四化表 → 飛星四化
```
