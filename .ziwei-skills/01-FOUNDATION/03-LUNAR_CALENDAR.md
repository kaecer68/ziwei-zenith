## 1. 農曆基礎

### 1.1 陰曆 vs 陽曆

| 類型 | 計算方式 | 用途 |
|------|---------|------|
| 陽曆 | 以太陽為準 | 24節氣、回歸年 |
| 陰曆 | 以月亮為準 | 月份、日期 |
| 陰陽合曆 | 兩者結合 | 紫微斗數使用 |

### 1.2 月份地支對照（固定不變）

| 月份 | 地支 | 節氣範圍 |
|:---:|:----:|:---------|
| 正月 | 寅 | 立春 → 驚蟄 |
| 二月 | 卯 | 驚蟄 → 清明 |
| 三月 | 辰 | 清明 → 立夏 |
| 四月 | 巳 | 立夏 → 芒種 |
| 五月 | 午 | 芒種 → 小暑 |
| 六月 | 未 | 小暑 → 立秋 |
| 七月 | 申 | 立秋 → 白露 |
| 八月 | 酉 | 白露 → 寒露 |
| 九月 | 戌 | 寒露 → 立冬 |
| 十月 | 亥 | 立冬 → 大雪 |
| 冬月 | 子 | 大雪 → 小寒 |
| 臘月 | 丑 | 小寒 → 立春 |

### 1.3 日干計算

日干根據「年前一位，月起方法」計算。

## 2. 節氣（24節氣）

### 2.1 節氣與月份

```
春季：立春 → 雨水 → 驚蟄 → 春分 → 清明 → 穀雨
夏季：立夏 → 小滿 → 芒種 → 夏至 → 小暑 → 大暑
秋季：立秋 → 處暑 → 白露 → 秋分 → 寒露 → 霜降
冬季：立冬 → 小雪 → 大雪 → 冬至 → 小寒 → 大寒
```

### 2.2 紫微斗數中的節氣

| 節氣 | 意義 |
|------|------|
| 立春 | 新年起點（寅月開始） |
| 清明 | 進入辰月 |
| 芒種 | 進入午月 |
| 白露 | 進入申月 |
| 立冬 | 進入亥月 |
| 大雪 | 進入子月 |

### 2.3 中氣

| 月份 | 中氣 |
|:---:|:----:|
| 正月 | 雨水 |
| 二月 | 春分 |
| 三月 | 穀雨 |
| 四月 | 小滿 |
| 五月 | 夏至 |
| 六月 | 大暑 |
| 七月 | 處暑 |
| 八月 | 秋分 |
| 九月 | 霜降 |
| 十月 | 小雪 |
| 冬月 | 冬至 |
| 臘月 | 大寒 |

## 3. 流月計算

### 3.1 流月地支

流月地支 = 根據當年立春後的月份

### 3.2 流月天干

流月天干根據「流年干 + 流月支」推算：

| 流年天干 | 正月天干 |
|:-------:|:-------:|
| 甲、己 | 丙 |
| 乙、庚 | 戊 |
| 丙、辛 | 庚 |
| 丁、壬 | 壬 |
| 戊、癸 | 甲 |

---

## 4. 流日計算

流日以每天的「子」時為分界。
