## 1. 十二宮名稱與順序

紫微斗數的十二宮如命盤所示，固定順序如下：

```
            巳           午           未           申
        ┌─────────┬─────────┬─────────┬─────────┐
        │  父母宮  │  福德宮  │  田宅宮  │  官祿宮  │
        │ (田宅宮) │ (事業宮) │  (家)   │ (事業)  │
        ├─────────┼─────────┼─────────┼─────────┤
        │  辰       │         │         │  酉      │
        │  命宮    │   【    │   】    │  交友宮  │
        │ (命)    │  中間   │         │ (僕役)  │
        ├─────────┼─────────┼─────────┼─────────┤
        │  卯       │         │         │  戌      │
        │  子女宮  │   宮    │         │  疾厄宮  │
        ├─────────┼─────────┼─────────┼─────────┤
        │  財帛宮  │  夫妻宮  │  兄弟宮  │  遷移宮  │
        │ (財)    │ (夫)    │ (兄)    │ (遷移)  │
        └─────────┴─────────┴─────────┴─────────┘
            寅           丑           子           亥
```

### 十二宮順序（命宮為起點）

| 順序 | 宮名 | 簡稱 | 意義 |
|:---:|:----:|:---:|:-----|
| 1 | 命宮 | 命 | 本人、性格、命主 |
| 2 | 兄弟宮 | 兄 | 兄弟姐妹、財庫 |
| 3 | 夫妻宮 | 夫 | 配偶、婚姻 |
| 4 | 子女宮 | 子 | 子孫、晚輩 |
| 5 | 財帛宮 | 財 | 財運、賺錢能力 |
| 6 | 疾厄宮 | 疾 | 健康、疾病 |
| 7 | 遷移宮 | 遷 | 外出、旅遊、人際 |
| 8 | 僕役宮 | 僕 | 下屬、朋友、人際 |
| 9 | 官祿宮 | 官 | 事業、學業、工作 |
| 10 | 田宅宮 | 田 | 房產、家業、固定資產 |
| 11 | 福德宮 | 福 | 福氣、享受、精神 |
| 12 | 父母宮 | 父 | 父母、長輩、文書 |

## 3. 身宮

身宮是另一個重要宮位，表示肉身、實際的自己。

| 身宮所在 | 特點 |
|---------|------|
| 與命宮同宮 | 身心合一 |
| 在夫妻宮 | 較重視感情 |
| 在官祿宮 | 事業心重 |
| 在財帛宮 | 實際、務實 |
| 在遷移宮 | 奔波操勞 |

---

## 4. 來因宮

來因宮是出生年干所在的宮位，代表「起因」。

- 甲年 → 廉貞在命宮
- 乙年 → 天機在命宮
- ...

來因宮用於飛星四化的計算起點。

---

## 5. 命宮的確定

### 5.1 以出生地支為準

命宮的地支 = 出生時的地支

| 出生時的地支 | 命宮地支 |
|:-----------:|:---------:|
| 子 | 子上 |
| 丑 | 丑上 |
| 寅 | 寅上 |
| ... | ... |

### 5.2 宮位流轉原則

> **重要**：命宮地支固定，但**宮名**隨命宮位置而流轉。

實例：
- 命宮地支是「辰」→ 辰宮 = 命宮
- 下一個宮（巳宮）= 兄弟宮
- 再下一個宮（午宮）= 夫妻宮
- ...
