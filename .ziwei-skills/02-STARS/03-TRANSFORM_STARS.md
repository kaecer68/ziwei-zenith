## 1. 四化的意義

| 四化 | 意義 | 特點 |
|:---:|:----:|:-----|
| 化祿 | 財祿 | 錢財、資源、享受 |
| 化權 | 權力 | 能力、執著、變動 |
| 化科 | 科名 | 智慧、名聲、才學 |
| 化忌 | 忌仇 | 糾紛、阻礙、忌恨 |

### 四化記憶

```
祿 → 資源、錢財（得）
權 → 能力、執著（用力）
科 → 智慧、名聲（有用）
忌 → 阻礙、糾紛（阻礙）
```

## 3. 兩種四化

### 3.1 生年四化

出生年份的天干決定，稱為「生年干四化」。

**計算公式**：
```
生年四化 = 出生年干 → 四化表 → 該星曜化為某化
```

### 3.2 飛星四化（宮干四化）

每個宮位有自己的「宮干」，宮干決定該宮位的四化飛星。

**計算公式**：
```
飛星四化 = 宮干 → 四化表 → 該星曜化為某化 → 飛入該星所在宮位
```

---

## 4. 飛星四化計算流程

### 步驟

1. 取得宮干（該宮位的天干）
2. 查四化表（天干 → 祿權科忌）
3. 找到被激活的星曜在何宮
4. 記錄該宮位的四化狀態

### 示例

假設「命宮」的宮干是「丙」：

1. 丙 → 四化表 → 天同化祿、天機化權、文昌化科、廉貞化忌
2. 假設「天同」星在「財帛宮」→ 財帛宮有「天同化祿」
3. 假設「天機」星在「官祿宮」→ 官祿宮有「天機化權」
4. ...
