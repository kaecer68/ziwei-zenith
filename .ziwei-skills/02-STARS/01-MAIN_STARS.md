## 1. 十四顆主星

主星是紫微斗數中最重要的星曜，共有14顆。

### 1.1 北斗主星（6顆）

| 星名 | 五行 | 特點 |
|:---:|:---:|:-----|
| 紫微星 | 土 | 帝王之星，最高位 |
| 天機星 | 木 | 智慧之星，謀略 |
| 太陽星 | 火 | 正義之星，事業 |
| 武曲星 | 金 | 財星，剛毅 |
| 天同星 | 水 | 福星，温和 |
| 廉貞星 | 火 | 次桃花，複雜 |

### 1.2 南斗主星（6顆）

| 星名 | 五行 | 特點 |
|:---:|:---:|:-----|
| 天府星 | 土 | 財庫，保守 |
| 太陰星 | 水 | 財星，温柔 |
| 貪狼星 | 木 | 桃花，慾望 |
| 巨門星 | 水 | 是非，口才 |
| 天相星 | 水 | 印星，理財 |
| 天梁星 | 土 | 蔭星，庇佑 |

### 1.3 南斗助星（2顆）

| 星名 | 五行 | 特點 |
|:---:|:---:|:-----|
| 七殺星 | 火 | 衝動，肅殺 |
| 破軍星 | 水 | 耗損，變動 |

## 2. 主星五行局

主星根據五行分成五種局：

| 五行局 | 主星 | 口訣 |
|:------:|:----:|:-----|
| 水二局 | 紫微天府太陰貪狼巨門天相破軍 | 水二局 |
| 木三局 | 天機天梁天同 | 木三局 |
| 金四局 | 武曲太陽 | 金四局 |
| 土五局 | 廉貞天相 | 土五局 |
| 火六局 | 紫微天府太陰貪狼巨門七殺 | 火六局 |

### 五行局記憶

```
水：太陰、貪狼、巨門、破軍
木：天機、天梁、天同
金：武曲、太陽
土：廉貞、天相、紫微、天府
火：七殺
```

## 3. 主星亮度（廟旺平陷）

每顆主星在12地支有不同的亮度：

### 亮度等級

| 等級 | 說明 | 備註 |
|:---:|:----:|:-----|
| 廟 | 最亮 | 星曜最強 |
| 旺 | 明亮 | 星曜強 |
| 得 | 中等 | 星曜普通 |
| 利 | 尚可 | 星曜偏弱 |
| 平 | 普通 | 星曜無力 |
| 陷 | 暗淡 | 星曜最弱 |

## 4. 主星組合

| 組合 | 效果 |
|------|------|
| 紫微+天機 | 智謀雙全 |
| 紫微+天府 | 保守穩重 |
| 紫微+貪狼 | 慾望強 |
