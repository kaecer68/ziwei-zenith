## 1. 輔助星分類

輔助星用於補充主星，分為以下類別：

### 1.1 六吉星

| 星名 | 特點 |
|:---:|:-----|
| 左輔星 | 助力，貴人 |
| 右弼星 | 助力，貴人 |
| 文昌星 | 文學，功名 |
| 文曲星 | 口才，藝術 |
| 天魁星 | 正面貴人 |
| 天鉞星 | 正面貴人 |

### 1.2 六煞星

| 星名 | 特點 |
|:---:|:-----|
| 擎羊星 | 刑災，衝突 |
| 陀羅星 | 糾纏，遲滯 |
| 火星 | 衝動，急性 |
| 鈴星 | 隱匿，固執 |
| 地空星 | 空亡，變動 |
| 地劫星 | 耗損，風險 |

### 1.3 其他輔星

| 星名 | 特點 |
|:---:|:-----|
| 祿存星 | 財祿，積蓄 |
| 天空星 | 空想，落空 |
| 台輔星 | 輔助，表態 |
| 封詰星 | 輔助，護衛 |
| 龍池星 | 水中，遷移 |
| 鳳閣星 | 高樓，文學 |

---

## 2. 輔助星記憶

### 2.1 六吉速記

```
左輔、右弼 → 雙輔
文昌、文曲 → 雙文
天魁、天鉞 → 雙魁
```

### 2.2 六煞速記

```
擎羊、陀羅 → 羊陀
火星、鈴星 → 火鈴
地空、地劫 → 地空劫
```
