## 1. 亮度等級（6級）

| 等級 | 說明 | 權重 |
|:---:|:----:|:----:|
| 廟 | 最旺，力量完全發揮 | +4 |
| 旺 | 力量強旺 | +3 |
| 得 | 力量普通 | +2 |
| 利 | 力量較弱 | +1 |
| 平 | 力量平平 | 0 |
| 陷 | 力量微弱或無力 | -2 |

### 4.2 Go 定義

```go
// 亮度枚舉
type Brightness int
const (
    BrightnessMing Brightness = iota // 廟
    BrightnessWang                   // 旺
    BrightnessDe                     // 得
    BrightnessLi                     // 利
    BrightnessPing                   // 平
    BrightnessXian                   // 陷
    BrightnessBu                     // 不
)

// 主星亮度表
var StarBrightnessTable = map[Star][]Brightness{
    StarZiwei: {W, W, D, W, M, D, W, M, D, P, M, D},
    // ... 其他主星
}

// 六吉星亮度表
var AuspiciousStarBrightnessTable = map[AuspiciousStar][]Brightness{
    AuspiciousZuofu: {W, M, D, W, M, D, W, M, D, W, M, D},
```

### 4.3 API 數據傳遞

後端通過 `assistant_star_details` 欄位傳遞輔星和煞星的亮度資訊：

```go
// gRPC / REST 響應結構
assistantStarDetails := make([]PalaceStar, 0)
for _, s := range chart.AssistantStars[b] {
    switch star := s.(type) {
    case basis.AuspiciousStar:
        brightness = basis.AuspiciousBrightnessLevel(star, b).String()
    case basis.LuCunStar:
        brightness = basis.LuCunBrightnessLevel(star, b).String()
    case basis.MaleficStar:
        brightness = basis.MaleficBrightnessLevel(star, b).String()
    }
    assistantStarDetails = append(assistantStarDetails, PalaceStar{
        Name:       starName,
        Brightness: brightness,
    })
}
```

### 4.4 前端顯示

前端使用 `assistant_star_details` 獲取所有星曜（包括輔星、煞星）的亮度：

```typescript
// 從 API 響應中獲取
interface PalaceData {
  star_details?: Array<{ name: string; brightness?: string }>;      // 主星
  assistant_star_details?: Array<{ name: string; brightness?: string }>; // 輔星、煞星
}

// 顯示時
{entry.assistant_star_details?.map((star) => (
  <div>
    {star.name} {star.brightness}
  </div>
))}
```
