## 1. 亮度等級（6級）

| 等級 | 說明 | 權重 |
|:---:|:----:|:----:|
| 廟 | 最旺，力量完全發揮 | +4 |
| 旺 | 力量強旺 | +3 |
| 得 | 力量普通 | +2 |
| 利 | 力量較弱 | +1 |
| 平 | 力量平平 | 0 |
| 陷 | 力量微弱或無力 | -2 |

## 4. 代碼對照

| 實體 | 程式碼位置 |
|------|-----------|
| 亮度表 | `pkg/basis/brightness.go` |
| 亮度枚舉 | `pkg/basis/brightness.go` | `Brightness` |
| UI 顏色 | `web/src/components/PalaceDetailView.tsx` | `getBrightnessColor()` |

### Go 定義

```go
type Brightness int
const (
    BrightnessMing Brightness = iota // 廟
    BrightnessWang                  // 旺
    BrightnessDe                    // 得
    BrightnessLi                    // 利
    BrightnessPing                  // 平
    BrightnessXian                 // 陷
)
```
