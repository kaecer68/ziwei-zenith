## 1. 三方四正的組成

### 三方

| 關係 | 說明 | 與本宮的距離 |
|------|------|:------------:|
| 本宮 | 核心宮位 | 0 |
| 三合宮 1 | 三合之一 | +4 或 -8 |
| 三合宮 2 | 三合之二 | +8 或 -4 |
| 對宮 | 對面宮位 | +6 或 -6 |

### 四正

四正 = 三方 + 對宮 = 7個宮位

## 3. 三方四正計算公式

### 3.1 宮位索引

```javascript
const palaceIndex = palaceCycle.indexOf(palaceName);
// palaceCycle = ['命宮', '兄弟宮', '夫妻宮', '子女宮', '財帛宮', '疾厄宮',
//                 '遷移宮', '僕役宮', '官祿宮', '田宅宮', '福德宮', '父母宮']
```

### 3.2 三方四正集合

```javascript
const getSanFangSet = (palaceName) => {
  const index = palaceCycle.indexOf(palaceName);
  if (index === -1) return new Set();
  
  return new Set([
    palaceCycle[index],           // 本宮
    palaceCycle[(index + 4) % 12], // 三合宮 1
    palaceCycle[(index + 6) % 12], // 對宮
    palaceCycle[(index + 8) % 12], // 三合宮 2
  ]);
};
```

## 4. 三方四正的論命意義

### 4.1 本宮

代表該主題的核心。

### 4.2 三合宮

三個宮位互相呼應，論命時要一併考量。

| 三合組合 | 意義 |
|---------|------|
| 命宮 + 財帛宮 + 官祿宮 | 命運、事業、財富 |
| 夫妻宮 + 遷移宮 + 僕役宮 | 婚姻、人際、外出 |
| 子女宮 + 田宅宮 + 福德宮 | 子孫、家業、享受 |

### 4.3 對宮

對宮是衝射位，看該主題的對立面或外在表現。

| 對宮 | 論命重點 |
|------|---------|
| 命宮的對宮是遷移宮 | 本人與外出的關係 |
| 夫妻宮的對宮是官祿宮 | 配偶與事業的關係 |
| 財帛宮的對宮是疾厄宮 | 財富與健康的關係 |
