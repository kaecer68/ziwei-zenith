## 2. 排盤步驟

### Step 1: 確定命宮地支

根據出生**時辰地支**推算命宮地支：

| 時辰 | 命宮地支 |
|:----:|:---------:|
| 子 | 子上 |
| 丑 | 丑上 |
| 寅 | 寅上 |
| 卯 | 卯上 |
| 辰 | 辰上 |
| 巳 | 巳上 |
| 午 | 午上 |
| 未 | 未上 |
| 申 | 申上 |
| 酉 | 酉上 |
| 戌 | 戌上 |
| 亥 | 亥上 |

> **原理**：命宮地支 = 出生時辰地支（固定對照）

### Step 2: 排列十二宮

從命宮地支出發，按順時針方向排列：

```
命宮（已知地支）
↓
兄弟宮（+1）
↓
夫妻宮（+2）
↓
子女宮（+3）
↓
財帛宮（+4）
↓
疾厄宮（+5）
↓
遷移宮（+6，對宮）
↓
僕役宮（+7）
↓
官祿宮（+8）
↓
田宅宮（+9）
↓
福德宮（+10）
↓
父母宮（+11）
```

### Step 3: 安主星

根據**出生年**的天干和五行局，安放14顆主星。

#### 3.1 確定五行局

| 年干 | 五行局 |
|:----:|:-------:|
| 甲、己 | 火六局 |
| 乙、庚 | 金四局 |
| 丙、辛 | 水二局 |
| 丁、壬 | 木三局 |
| 戊、癸 | 土五局 |

#### 3.2 五行局定局表

| 五行局 | 口訊 |
|:------:|:-----|
| 水二局 | 太陰、貪狼、巨門、破軍 |
| 木三局 | 天機、天梁、天同 |
| 金四局 | 武曲、太陽 |
| 土五局 | 廉貞、天相、紫微、天府 |
| 火六局 | 七殺 |

#### 3.3 起星法

以**紫微星**為例：

```
紫微星 → 按五行局尋找
水二局：太陰宮起
木三局：天機宮起
金四局：武曲宮起
土五局：廉貞宮起
火六局：紫微宮起
```

### Step 4: 安輔助星

根據主星位置和年干，安放六吉星、六煞星等。

### Step 5: 計算生年四化

根據**出生年干**，查四化表：

```
生年四化 = 出生年干 → 四化表 → 該星曜化為某化
```
